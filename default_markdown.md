# 歡迎使用 Markdown 編輯器

這是一個可以**即時預覽**並複製為**富文本**的工具。

## 功能特點：
1. **自動儲存**：重新整理網頁也不會不見。
2. **富文本複製**：按右上角按鈕，可直接貼到 Gmail/Word。
3. **簡單乾淨**：專注於內容寫作。

> 試著在這裡修改看看吧！

## 快速上手
- 在左側輸入 Markdown
- 在右側即時查看結果
- 點擊「複製富文本」按鈕

## 範例表格
| 功能 | 支援 | 備註 |
| :--- | :---: | :--- |
| 即時預覽 | ✅ | 打字即轉換 |
| 自動儲存 | ✅ | 瀏覽器快取 |
| 富文本複製 | ✅ | 適合貼至 Word/Gmail |
